Implement the Channel class so that it will be used as a communication channel between producer and consumer.

Implement take() method by:
  1. make method synchronized so that it will be safe to use in multi-threaded environment
  2. check if channel is empty and then callWaitMethod()
  3. if channel is not empty, then:
    1. return the message
    2. mark channel as empty
    3. notify listeners about changes in the channel using notifyListenersAboutChangesInChannel()
Implement put() method by:
  1. make method synchronized so that it will be safe to use in multi-threaded environment
  2. check if channel is full and then callWaitMethod()
  3. if channel is empty, then:
    1. put the message in the channel
    2. mark channel as full
    3. notify listeners about changes in the channel using notifyListenersAboutChangesInChannel()
